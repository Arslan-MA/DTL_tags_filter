Теги

Тег verbatim

Останавливает механизм шаблонов от рендеринга содержимого этого тега блока. Часто используется для того, чтобы разрешить слой шаблонов JavaScript, который сталкивается с синтаксисом Django.

Синтаксис

{% verbatim %}
  {{ value }}
{% endverbatim %}}

Пример без тега verbatim:

{{ value }}

Пример с тегом verbatim:

{% verbatim %} {{ value }} {% endverbatim %}

Тег debug

Выводит целую кучу отладочной информации, включая текущий контекст и импортированные модули. {% debug %} ничего не выводит, если параметр DEBUG равен False.

Синтаксис

{% verbatim %} {% debug %} {% endverbatim %}

Пример без тега debug:

Пример с тегом debug:

Тег include

Загружает шаблон и отображает его с текущим контекстом. Это способ «включения» других шаблонов в шаблон. Имя шаблона может быть переменной или жестко закодированной (заключенной в кавычки) строкой в одинарных или двойных кавычках.

Синтаксис

{% verbatim %} {% include "app/inc.html" %} {% endverbatim %}

Пример без тега include:

My code....

Пример с тегом include:

My code.... {% include "app/urlize.html" %}

Тег ifchanged

Проверьте, изменилось ли значение с последней итерации цикла.

Синтаксис

{% verbatim %} {% ifchanged %} ... {% endifchanged %} {% endverbatim %}

Пример без тега ifchanged:

{% for food in foods %}

{{ food.category }}

{{ food.name }}

{% endfor %}

Пример с тегом ifchanged:

{% for food in foods %} {% ifchanged food.category %}

{{ food.category }}

{% endifchanged %}

{{ food.name }}

{% endfor %}

Тег load

Загружает настраиваемый набор тегов шаблона.

Синтаксис

{% verbatim %} {% load static %} {% endverbatim %}

Пример без тега load:

my block

Пример с тегом load:

my block

Тег with

Кэширует сложную переменную под более простым именем. Это полезно при многократном доступе к «дорогостоящему» методу (например, тот, который делает запрос в базу данных).

Синтаксис

{% verbatim %} {% with firstname="Stalikken" %} .... {% endwith %} {% endverbatim %}

Пример без тега with:

Hello

Пример с тегом with:

{% with firstname="Stalikken" %} Hello {{ firstname }} {% endwith %}

Тег autoescape

Управляет текущим автоматическим экранированием. Этот тег принимает в качестве аргумента либо on, либо off и определяет, действует ли автоматическое экранирование внутри блока. Блок закрывается конечным тегом endautoescape.

Синтаксис

{% verbatim %} {% autoescape on %} {{ blurb }} {% endautoescape %}} {% endverbatim %}

Пример без тега autoescape:

{{ blurb }}

Пример с тегом autoescape:

{% autoescape off %} {{ blurb }} {% endautoescape %}

Тег lorem

Отображает случайный латинский текст «lorem ipsum». Это полезно для предоставления образцов данных в шаблонах.

Синтаксис

{% verbatim %} {% autoescape on %} {% lorem 2 w random %} {% endautoescape %}} {% endverbatim %}

Пример без тега lorem:

Пример с тегом lorem:

{% lorem 2 w random %}

Фильтры

Фильтр force_escape

Применяет экранирование HTML к строке. Этот фильтр применяется немедленно и возвращает новую строку с экранированием. Это полезно в редких случаях, когда требуется многократное экранирование или вы хотите применить другие фильтры к результатам экранирования.

Синтаксис

{% verbatim %} {{ value|linebreaks|force_escape }} {% endverbatim %}

Пример без фильтра title:

{{ value }}

Пример с фильтром title:

{{ value|linebreaks|force_escape }}

Фильтр linebreaksbr

Принимает список словарей и возвращает этот список, отсортированный в обратном порядке по ключу, указанному в аргументе.

Синтаксис

{% verbatim %} {% for food in foods|dictsortreversed:'category' %}
  {{ food.name }}, {{ food.category }}
{% endfor %} {% endverbatim %}

Пример без фильтра dictsortreversed:

    {% for food in foods %}
  • {{ food.name }}, {{ food.category }}
    • {% endfor %}

Пример с фильтром dictsortreversed:

    {% for food in foods|dictsortreversed:'category' %}
  • {{ food.name }}, {{ food.category }}
    • {% endfor %}

Фильтр linebreaksbr

Преобразует все новые строки в фрагменте обычного текста в разрывы строк HTML (br).

Синтаксис

{% verbatim %} {{ value|linebreaksbr }} {% endverbatim %}

Пример без фильтра linebreaksbr:

{{ value }}

Пример с фильтром linebreaksbr:

{{ value|linebreaksbr }}

Фильтр urlize

Преобразует URL-адреса и адреса электронной почты в тексте в кликабельные ссылки.

Синтаксис

{% verbatim %} {{ url|urlize }} {% endverbatim %}

Пример без фильтра urlize:

{{ url }}

Пример с фильтром urlize:

{{ url|urlize }}

Фильтр safeseq

Применяет фильтр safe к каждому элементу последовательности. Используется в сочетании с другими фильтрами, работающими с последовательностями, например join.

Синтаксис

{% verbatim %} {{ foods|safeseq|join:", " }} {% endverbatim %}

Пример без фильтра safeseq:

    {{ foods }}

Пример с фильтром safeseq:

{{ foods|safeseq|join:"
" }}

Фильтр default

Если значение равно False, используется заданное значение по умолчанию. В противном случае используется значение.

value_default : пустая строка

Синтаксис

{% verbatim %} {{ value_default|default:"nothing" }} {% endverbatim %}

Пример без фильтра default:

{{ value_default }}

Пример с фильтром default:

{{ value_default|default:"nothing" }}

Фильтр safe

Помечает строку как не требующую дальнейшего экранирования HTML перед выводом. Когда автоэскейпинг выключен, этот фильтр не действует.

Синтаксис

{% verbatim %} {{ blurb|safe }} {% endverbatim %}

Пример без фильтра safe:

{{ blurb }}

Пример с фильтром safe:

{{ blurb|safe }}

Фильтр unordered_list

Рекурсивно принимает самовложенный список и возвращает неупорядоченный список HTML – БЕЗ открывающих и закрывающих тегов ul.

Синтаксис

{% verbatim %} {{ var|unordered_list }} {% endverbatim %}

Пример без фильтра unordered_list:

{{ var }}

Пример с фильтром unordered_list:

{{ var|unordered_list }}

Фильтр truncatewords_html

Аналогичен truncatewords, за исключением того, что он знает о тегах HTML. Любые теги, открытые в строке и не закрытые до точки усечения, закрываются сразу после усечения.

Синтаксис

{% verbatim %} {{ value|truncatewords_html:2 }} {% endverbatim %}

Пример без фильтра truncatewords_html:

{{ value }}

Пример с фильтром truncatewords_html:

{{ value|truncatewords_html:2 }}

Синтаксис

Пример без фильтра rjust:

Пример с фильтром rjust:

Фильтр filesizeformat

Форматирует значение как «человекочитаемый» размер файла (т.е. '13 KB', '4.1 MB', '102 bytes' и т.д.).

Синтаксис

{% verbatim %} {{ value_num|filesizeformat }} {% endverbatim %}

Пример без фильтра filesizeformat:

{{ value_num }}

Пример с фильтром filesizeformat:

{{ value_num|filesizeformat }}

Фильтр wordwrap

Обертывает слова на заданную длину строки.

Синтаксис

{% verbatim %} {{ text|wordwrap:10 }} {% endverbatim %}

Пример без фильтра wordwrap:

{{ text }}

Пример с фильтром wordwrap:

{{ text|wordwrap:10|linebreaksbr }}

Пример без фильтра join:

{{ list }}

Пример с фильтром join:
